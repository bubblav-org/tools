# BubblaV Claude Code Plugin

Install directly from GitHub:

```text
/plugin install github:bubblav-org/tools/claude-code-plugin
```

This plugin includes:

- MCP configuration (`.mcp.json`) for `https://www.bubblav.com/api/mcp`
- Skill `/bubblav-tools:web-scrape-md`
- Tool support for `bubblav_scrape_url`

## Configure API key

Set your API key before launching Claude Code:

```bash
export BUBBLAV_API_KEY=bubblav_mcp_YOUR_API_KEY
```
